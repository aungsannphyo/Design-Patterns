# Design Patterns

Design Patterns တွေကိုအလွယ်ကူဆုံးနဲ့အရိုးရှင်းဆုံးဖစ်အောင် Java ကိုအသုံးပြုပီး ကျွန်တော်နားလည်တက်ကျွမ်းသလောက် ရေးသားတင်ဆက် ပေးသွားမှာဖစ်ပါတယ်ခင်ဗျာ။
